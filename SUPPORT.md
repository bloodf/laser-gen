# Support

Thanks for using laser-gen! Here's how to get help.

## Questions and usage help

- **Documentation first**: check the [README](README.md) and the [docs/](docs/) folder —
  especially [wrap math](docs/wrap-math.md) and the guides for
  [adding a vessel](docs/adding-a-vessel.md) and [adding a language](docs/adding-a-language.md).
- **GitHub Discussions**: for open-ended questions, ideas, and show-and-tell, use
  [Discussions](https://github.com/bloodf/laser-gen/discussions).

## Bugs and feature requests

Please use the issue templates so we get the details we need:

- [Bug report](../../issues/new?template=bug_report.yml)
- [Feature request](../../issues/new?template=feature_request.yml)
- [Vessel request](../../issues/new?template=vessel_request.yml)
- [Translation report](../../issues/new?template=translation_report.yml)

## Security issues

Never report security issues publicly — see [SECURITY.md](SECURITY.md).

## What we can't help with

- Support for third-party laser software (LightBurn, xTool Creative Space, LaserGRBL) —
  please use their respective support channels.
- Recovering data deleted from your browser storage. laser-gen is fully client-side; your
  browser is the only place your projects exist. Export important work regularly.
